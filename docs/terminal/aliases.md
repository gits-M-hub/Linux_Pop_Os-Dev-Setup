# Alias

Los alias permiten ejecutar comandos largos utilizando nombres cortos.

---

## Git

### gs

```bash
gs
```

Equivale a:

```bash
git status
```

---

### ga

```bash
ga
```

Equivale a:

```bash
git add
```

---

### gc

```bash
gc
```

Equivale a:

```bash
git commit
```

---

### gp

```bash
gp
```

Equivale a:

```bash
git push
```

---

### gl

```bash
gl
```

Equivale a:

```bash
git pull
```

## Alias del sistema

### cat

```bash
cat
```

Utiliza internamente:

```bash
batcat
```

para mostrar archivos con resaltado de sintaxis.
