# Navegación en Linux

## Saber dónde estoy

```bash
pwd
```

---

## Ir al directorio personal

```bash
cd
```

o

```bash
cd ~
```

---

## Cambiar de carpeta

```bash
cd Carpeta
```

---

## Volver atrás

```bash
cd ..
```

---

## Volver al directorio anterior

```bash
cd -
```

---

## Mostrar archivos

```bash
ls
```

---

## Mostrar información detallada

```bash
ls -lah
```

---

## Ver estructura del proyecto

```bash
tree
```

o

```bash
tree -L 2
```

---

## Buscar archivos

```bash
find . -name "*.md"
```

---

## Crear carpetas

```bash
mkdir nombre
```

Crear varias carpetas:

```bash
mkdir -p ruta/completa
```

---

## Crear archivos

```bash
touch archivo.txt
```

---

## Copiar

```bash
cp origen destino
```

---

## Mover o renombrar

```bash
mv origen destino
```

---

## Eliminar

```bash
rm archivo
```

---

## Recargar Zsh

```bash
source ~/.zshrc
```

---

## Verificar si un programa está instalado

```bash
command -v programa
```
