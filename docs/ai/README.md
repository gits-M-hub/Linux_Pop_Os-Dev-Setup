## 🧠 Filosofía de uso

Las herramientas de IA en este entorno se utilizan para:

- **Acelerar el desarrollo** - No reemplazar, sino potenciar
- **Aprender** - Entender el código generado
- **Automatizar tareas repetitivas** - Enfocarse en lo importante
- **Mejorar calidad** - Code review asistido
- **Documentar** - Generación de documentación

**Principios:**
- Siempre revisar el código generado
- Entender antes de usar
- No depender ciegamente de la IA
- Usar como asistente, no como reemplazo
