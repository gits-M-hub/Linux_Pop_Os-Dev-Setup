## 💡 Tips para mejores prompts

- **Sé específico** - Cuanto más específico, mejor el resultado
- **Proporciona contexto** - Incluye código relevante y descripciones
- **Especifica el formato** - Indica el formato de salida deseado
- **Itera** - Refina el prompt si la primera respuesta no es ideal
- **Usa ejemplos** - Proporciona ejemplos del estilo deseado
